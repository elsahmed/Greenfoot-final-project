USER INSTRUCTIONS:
  1. Hit the [Space bar] to go to instructions page
    - Hit [Enter] to go straight to the offical game
      --> Hit [Enter] in the beginning if user wants to skip instructions page
  2. Use the "up", "left" & "right" arrow keys to move the rocketship around the globe
  3. OBJECTIVE:
    - Collect as many coins as you can before getting hit by 5 Aliens
      * A healthbar at the bottom of the screen will let you know if you are running 
        low on health 
    - To gain more energy try catching a star
  4. The higher the level the user is on the faster and more quantities of Aliens will appear
  5. To exit the game at anytime hit [Esc]
  6. Once game is over the user can hit the [H] to go back to the home screen
